Statoshi: Bitcoin ABC Core + statistics logging
=====================================

What is Statocash ?
----------------

Statocash is a fork of JLOPP Statoshi http://statoshi.info) adapted to work with
bitcoin cash. It use bitcoin abc core and all stuff from statoshi.
